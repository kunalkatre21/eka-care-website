Animated Borders Demo

Showcase of animated border effects for Eka Healthcare components. These styles can be applied to any element using the provided CSS classes.

Gradient Border

Rotating gradient border with healthcare color scheme

Glowing Border

Animated glowing effect with blur for premium feel

Pulse Border

Heartbeat-style pulsing animation for medical themes

Glimmer Border

Subtle rotating glimmer effect around the border

SVG Stroke

Animated SVG stroke drawing effect

Rotating Border

Continuous rotation with conic gradient

How to Use

Basic Usage

<div class="animated-border-gradient">
    <!-- Your content here -->
</div>

Available Classes

  • .animated-border-gradient - Rotating gradient
  • .animated-border-glow - Glowing effect
  • .animated-border-pulse - Pulse animation
  • .animated-border-glimmer - Rotating glimmer
  • .animated-border-svg - SVG stroke
  • .animated-border-rotating - Conic rotation

Advanced Usage with DaisyUI

<div class="card animated-border-gradient shadow-xl">
    <div class="card-body">
        <h2 class="card-title">Healthcare Feature</h2>
        <p>Enhanced with animated borders</p>
        <div class="card-actions justify-end">
            <button class="btn btn-primary">Action</button>
        </div>
    </div>
</div>

Customization

All animations respect the prefers-reduced-motion media query for accessibility. Colors are optimized for healthcare applications with blue, green, and purple themes.

These styles are included in shared-styles.css and can be used across all healthcare components.
